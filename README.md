# llm-cal 用户手册

`llm-cal` 是一个基于命令行终端的 AI 日历助手，它将强大的自然语言大模型（Claude）与经典的 Unix 终端日历工具 `calcurse` 完美结合。让你可以在终端里“用人话”管理一切行程和待办。

---

## 1. 核心概念

`llm-cal` 不存储任何私有格式的数据，它本质上是一个 **“翻译官”**。
当你输入一句话，它会：
1. 读取当前的日期、现有的日程和待办。
2. 调用 AI 理解你的意图（增、删、改、清空）。
3. 将你的话精准翻译成 `calcurse` 的纯文本数据结构。
4. 自动写入 `~/.local/share/calcurse/` 中的 `apts` (日程) 或 `todo` (待办) 文件。

这使得你可以**随时**打开标准的 `calcurse` 界面去查看这些通过 AI 生成的内容。

---

## 2. 功能与使用示例

你只需要在终端输入 `llm-cal` 加上你的指令即可。支持批量和混合指令。

### 🗓️ 管理日程 (Calendar)

*   **添加日程**
    *   `llm-cal "明天下午3点和老王开会"` -> *(自动计算明天的日期并设定为 15:00-16:00)*
    *   `llm-cal "下周三早上九点半去机场接人，大概需要两个小时"`
*   **修改/推迟日程**
    *   `llm-cal "把明天和老王的开会推迟到后天下午4点"`
    *   `llm-cal "机场接人改到下周四"`
*   **取消/删除日程**
    *   `llm-cal "帮我取消和老王的会"`

### ✅ 管理待办 (To-Do)

*   **添加待办任务**
    *   `llm-cal "提醒我买牛奶"` -> *(默认优先级)*
    *   `llm-cal "记一个最高优先级的待办：给老板发周报"` -> *(自动生成 [1] 级优先度)*
*   **完成/删除任务**
    *   `llm-cal "买牛奶那件事我已经做完了"`
    *   `llm-cal "划掉给老板发周报的待办"`

### 💥 批量与组合操作 (Multi-action)

AI 可以同时处理一句话中的多个需求：
*   `llm-cal "取消下午的开会，然后记一个明天去超市买机油的待办"`
*   `llm-cal "删除所有的日程和待办事"` -> *(危险操作：清空所有数据)*

## 3. 安全与备份恢复

因为自然语言和终端操作具有破坏性（比如一句话清空所有日程），`llm-cal` 内置了**自动防呆备份系统**。

**触发条件：** 当你执行任何 `修改`、`删除` 或 `清空` 操作前，系统会自动对现有数据进行快照备份。

**如何恢复？**
1. 输入命令查看最近的备份：
   ```bash
   llm-cal "撤销" 
   # 或者 llm-cal "备份"
   ```
2. 系统会输出类似如下的备份列表：
   ```text
   最近的备份文件 (存放在 ~/.local/share/calcurse/backups/):
     - todo_20260520_153022.bak
     - apts_20260520_153022.bak
   ```
3. 根据提示手动覆盖恢复：
   ```bash
   cp ~/.local/share/calcurse/backups/apts_20260520_153022.bak ~/.local/share/calcurse/apts
   ```

## 4. 技术栈与环境依赖

*   **底层应用:** `calcurse` (通过 `sudo apt install calcurse` 安装，数据存在 `~/.local/share/calcurse/`)
*   **运行环境:** `uv` (现代 Python 包管理器) 和 `Python 3`
*   **AI 引擎:** Anthropic API (目前使用的是 `claude-haiku-4-5-20251001` 模型，速度快且精准)
*   **执行文件位置:** `~/.local/bin/llm-cal` (你可以随时用 `vim` 编辑它修改提示词逻辑)
